# 对话框 - Dialog

在保留当前页面状态的情况下，告知用户并承载相关操作。

# 示例

温馨提示

123456
点击查看代码 
  <template>
      <div class="dialog">
          <div class="row">
              <pf-button type="primary" @click="visible = true">点击出现对话框</pf-button>
              <pf-dialog width="60%" top="20vh" :visible.sync="visible">
                  <template v-slot:title>
                      <h3>温馨提示</h3>
                  </template>
                  <div>123456</div>
                  <template v-slot:footer>
                      <pf-button @click="visible = false">取消</pf-button>
                      <pf-button type="primary" @click="visible = false">确定</pf-button>
                  </template>
              </pf-dialog>
          </div>
      </div>
  </template>
  
  <script>
      export default {
          name: "my-dialog",
          data(){
              return{
                  visible: false,
              }
          }
      }
  </script>

# API

参数名 参数描述 参数类型 默认值
title 对话框标题 string 提示
width 宽度 string 50%
top 与顶部的距离 string 15vh 或 200px
visible 是否显示dialog（支持sync修饰符） boolean false

# 事件支持：

事件名 事件描述
opened 模态框显示的事件
closed 模态框关闭的事件

# 插槽说明：

插槽名称 插槽描述
default dialog的内容
title dialog的标题
footer dialog的底部操作区
